8.1 定义函数

def greet_user(username):
    """显示简单的问候语"""
    print(f"Hello, {username.title()}!")
   
greet_user('jesse')

  • 位置实参

  • 关键字实参

    关键字实参是传递给函数的名称值对，直接在是惨重将名称和值关联起来。无需考虑函数调用中的实参顺序。

    def describe_pet(animal_type, pet_name)

8.3 返回值

def get_formatted_name(first_name, last_name):
    full_name = f"{first_name} {last_name}"
    return full_name
  • 返回字典
def build_person(first_name, last_name, age = None):
    person = {'first' : first_name, 'last' : last_name}
    if age:
        person['age'] = age
        return person

8.4 传递列表

def greet_users(names):

有一家3D打印的公司，未打印的模型打印完毕后移到打印好的模型列表中。要求保留原来未打印的设计列表以供备案。

使用切片表示法创建列表的副本：

def print_models(unprinted_designs[:], completed_models):
    # ...

Remark:除非有充分的理由，否则还是应该将原始列表传递给函数。用线程的列表可以避免花时间和内存创建副本。

8.5 传递任意数量的实参

def make_pizza(*toppings) :
    # ...

形参名 *toppings中的星号让 Python 创建一个名为 toppings 的空元组，并将收到的所有值都封装到这个元组中。

  • 位置实参和任意数量实参结合使用

    必须将函数定义中接纳任意数量实参的形参放在最后。

    def make_pizza(size, *toppings)
	# ...

  • 使用任意数量的关键字实参

def build_profile(first, last, **user_info):
    user_info['first_name'] = first
    user_info['last_name'] = last
    return user_info

user_profile = build_profile('albert', 'einstein', location = 'princeton', field = 'physics')

形参**user_info 中的两个星号让 python 创建一个名为 user_info 的空字典，并将收到的所有名称对都放到这个字典中。

8.6将函数存储在模块中

通过将函数存储在称为模块的独立文件中可以隐藏程序代码的细节，将重点放在程序的高层逻辑。

# pizza.py
def make_pizza(size, *toppings):
    # ...

# making_pizzas.py
import pizza
pizza.make_pizza(16, 'pepperoni')

代码行 import pizza 让 Python 打开文件 pizza.py,并将其中的所有函数都复制到这个程序中。

要调用被导入模块中的函数，可以使用下面的语法：

module_name.function_name()
  • 还可以导入模块中的特定函数
from module_name import function_0, function_1, function_2

导入模块中的特定函数后，调用函数时无须使用句点。

  • 使用 as 给函数指定别名

如果要导入函数的名称可能与程序中现有的名称冲突，或者函数的名称太长，可以指定一个别名。

from module_name import function_name as fn
  • 使用as给模块指定别名
import pizza as p
p.make_pizza(16, 'pepperoni')
  • 导入模块中的所有函数

使用星号(*)运算符可以让Python导入模块中的所有函数：

from pizza import *

此时可以通过名称来调用每个函数，而无须使用句点表示法。

第九章 类

9.1 创建和使用类

class Dog:
    def __init__(self, name, age):
        self.name = name
        self.age = age
   
	def sit(self) :
        # ...
   
my_dog = Dog('Willie', 6)
age = my_dog.name
my_dog.sit()

__ init__ 是一个特殊方法，创建Dog新实例时会自动运行。self必不可少，且必须位于其他形参的前面，Python调用这个方法创建Dog实例时，将自动传入实参self.

9.2 使用类和实例

创建实例时，有些属性无须通过形参来定义，可在方法 __ init __ ()中为其指定默认值。

class Car:
    def __init__(self, make, model, year):
        self.make = make
        self.model = model
        self.year = year
        self.odometer_reading = 0
  • 可以直接修改属性的值或者通过方法修改属性的值

9.3 继承

创建子类时，父类必须包含在当前文件中，且位于子类前面。

定义子类时，必须在圆括号中指定父类的名称。

class ElectricCar(Car):
    
    def __init__(self, make, model, year):
        super().__init__(make, model, year)
        self.battery_size = 75

super()是一个特殊函数，可以调用父类的方法。

  • 重写父类的方法

Car类有一个名为 fill_gas_tank()的方法，对电动汽车来说毫无意义。

clss ElectricCar(Car):
    --snip--
    def fill_gas_tank(self):
        print("This car doesn't need a gas tank!")
  • 将实例用作属性
class Battery:
    def __init__ (self, battery_size = 75):
        self.battery_size = battery_size
        # ...
    
class ElectricCar(Car):
    def __init__ (self, make, model, year):
        # ...
        self.battery = Battery()

9.4 导入类

from car import ElectricCar, Car

my_tesla = ElectricCar('tesla', 'model s', 2019)
  • 还可以导入整个模块，再使用句点表示法访问需要的类。
import car
my_beetle = car.Car('volkswagen', 'beetle', 2019)
my_tesla = car.ElectricCar('tesla', 'roadster', 2019)
  • 导入模块中所有的类
from module_name import *
  • 在一个模块中导入另一个模块

有时候可能一个模块中的类依赖于另一个模块中的类。在这种情况下，需要在模块中导入必要的类。

* 使用别名

from electric_car import Electric_Car as EC
my_tesla = EC('tesla', 'roadster', 2019)
