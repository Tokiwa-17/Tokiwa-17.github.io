Python编程：从入门到实践

2020-11-10

Python 编程从入门到实践

第二章 变量和简单数据类型

字符串

  1. 修改大小写

用引号扩起的都是字符串，其中的引号可以是单引号，也可以是双引号

"this is a string."
'this is a string.'
'I told my friend, "python is my favorite language!"'

title() 以首字母大写的方式显示每个单词（其余字母小写）

upper() 全部大写 lower() 全部小写.

  1. 在字符串中使用变量(f字符串)

python把花括号中的变量替换为其值来设置字符串的格式。

first_name = "ada"
last_name = "lovelace"
full_name = f"{first_name} {last_name}" # full_name = "ada lovelace"
  1. 制表符或换行符
print("\tPython")
print("Languages:\nPython\nC\nJavaScript")
  1. 删除空白

暂时删除空白

favorite_language = 'python '
favorite_language.rstrip() # 'python'
favorite_language # 'python '

永久删除空白

favorite_language = favorite_language.rstrip() # 'python'

剔除字符串开头的空白或者两边的空白可以分别使用 lstrip() 和 strip().

数

  1. python 使用两个乘号表示乘方运算：
>>> 3 ** 2
9

  1. 将任意两个数相除时，结果总是浮点数，即便这两个数都是整数且能整除。

  2. 书写很大的数时，可使用下划线将其中的数字分组使其更易读：

>>> universe_age = 14_000_000_000
>>> print(universe_age)
14000000000

  1. 同时给多个变量赋值

    >>> x, y, z = 0, 0, 0

    只要变量和值的个数相同，Python就能正确地把它们关联起来

  2. 常量

Python 中没有内置的常量类型，通常使用全大写来指出某个变量为常量。

MAX_CONNECTIONS = 5000

第三章：列表简介

列表是什么

一系列按特定顺序排列的元素组成列表 有序列表

bicycles = ['trek', 'cannondale', 'redline', 'specialized']
  1. 访问列表元素
print(bicycles[0]) 
# trek
print(bicycles[-1])
# specialized
