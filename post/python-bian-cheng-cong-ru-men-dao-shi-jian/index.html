2020-11-10

Python 编程从入门到实践

第二章 变量和简单数据类型

字符串

  1. 修改大小写

用引号扩起的都是字符串，其中的引号可以是单引号，也可以是双引号

"this is a string."
'this is a string.'
'I told my friend, "python is my favorite language!"'

title() 以首字母大写的方式显示每个单词（其余字母小写）

upper() 全部大写 lower() 全部小写.

  1. 在字符串中使用变量(f字符串)

python把花括号中的变量替换为其值来设置字符串的格式。

first_name = "ada"
last_name = "lovelace"
full_name = f"{first_name} {last_name}" # full_name = "ada lovelace"
  1. 制表符或换行符
print("\tPython")
print("Languages:\nPython\nC\nJavaScript")
  1. 删除空白

暂时删除空白

favorite_language = 'python '
favorite_language.rstrip() # 'python'
favorite_language # 'python '

永久删除空白

favorite_language = favorite_language.rstrip() # 'python'

剔除字符串开头的空白或者两边的空白可以分别使用 lstrip() 和 strip().

数

  1. python 使用两个乘号表示乘方运算：
>>> 3 ** 2
9

  1. 将任意两个数相除时，结果总是浮点数，即便这两个数都是整数且能整除。

  2. 书写很大的数时，可使用下划线将其中的数字分组使其更易读：

>>> universe_age = 14_000_000_000
>>> print(universe_age)
14000000000

  1. 同时给多个变量赋值

    >>> x, y, z = 0, 0, 0

    只要变量和值的个数相同，Python就能正确地把它们关联起来

  2. 常量

Python 中没有内置的常量类型，通常使用全大写来指出某个变量为常量。

MAX_CONNECTIONS = 5000

第三章：列表简介

列表是什么

一系列按特定顺序排列的元素组成列表 有序列表

bicycles = ['trek', 'cannondale', 'redline', 'specialized']
  1. 访问列表元素
print(bicycles[0]) 
# trek
print(bicycles[-1])
# specialized

Python可以在不知道列表长度的情况下访问最后的元素

  1. 修改列表元素
motorcycles = ['honda', 'yamaha', 'suzuki']
motorcycles.append('ducati')
print(motorcycles)
# ['honda', 'yamaha', 'suzuki', 'ducati']

实际中经常创建一个空列表，然后程序运行后动态加入列表元素。

使用方法insert()可以在任何位置添加新元素

motorcycles = ['honda', 'yamaha', 'suzuki']
motorcycles.insert(0, 'ducati')
# ['ducati', 'honda', 'yamaha', 'suzuki']

如果知道要删除的元素在列表中的位置，可使用del语句。

del motorcycles[0]

使用方法pop()删除元素：删除列表末尾的元素，并能够继续使用它

motorcycles = ['honda', 'yamaha', 'suzuki']
popped_motorcycles = motorcycles.pop()

实际上，可以用pop()来删除列表中任意位置的元素, 只需在圆括号中指定要删除元素的索引。

根据值删除元素

motorcycles = ['honda', 'yamaha', 'suzuki', 'ducati']
motorcycles.remove('ducati')
  • Remark: 方法Remove()只删除第一个指定的值，如果要删除的值可能在列表中出现多次，就需要使用循环来确保每个值都删除。
  1. 组织列表
  • 方法sort()永久性地修改列表元素的顺序
cars = ['bmw', 'audi', 'toyota' , 'subaru']
cars.sort() # ['audi', 'bmw', 'subaru', 'toyota']

还可以按反字母序的顺序排列列表元素

cars.sort(reverse = True)
# ['toyota', 'subaru', 'bmw', 'audi']

* 使用函数sorted()对列表临时排序

print(sorted(cars)) # ['audi', 'bmw', 'subaru', 'toyota']
print(cars) # ['bmw', 'audi', 'toyota', 'subaru']

方法reverse()：倒序排列

方法len(): 获取列表的长度
