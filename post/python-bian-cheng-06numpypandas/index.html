Python 编程06： Numpy、Pandas

Numpy、Pandas用底层语言（C）实现，比Python自带类型要快很多。

1.1 numpy属性

import numpy as np

列表转换成矩阵:

array = np.array([[1, 2, 3],
                  [2, 3, 4]])
# [[1 2 3]
#	[2 3 4]] 没有逗号分隔

print(array.shape) # (2, 3)
print(array.size)  # 6

1.2 numpy的创建array

  • 可以在创建的时候使用 dtype来指定元素类型：
a = np.array([2, 23, 4], dtype = np.float)
print(a.dtype) # float64
  • 生成零矩阵(提供元组形参)
a = np.zeros((3, 4))
  • 生成有序数列
a = np.arange(10, 20, 2) # (start, end, step)
  • 重新指定有序数列的格式
a = np.arrat(12).reshape((3, 4))
  • 生成线段
a = np.linspace(1, 10, 5)
# 在 1 ~ 10之间生成五段的线段

1.3 numpy的基础运算

a = np.array([10, 20, 30, 40])
b = np.arange(4)

c = a + b
c = a - b
c = b ** 2 # [0, 1, 4, 9]
c = 10 * np.sin(a)
print(b < 3) #[True True True False]
  • 矩阵运算
a = np.array([[1, 1],
              [0, 1]])
b = np.arange(4).reshape((2, 2))
# 逐个相乘
c = a * b
# 矩阵乘法
c_dot = np.dot(a, b)
c_dot = a.dot(b)

a = np.random.random((2, 4)) #shape
print(np.sum(a))
# axis = 1 每一行
# axis = 0 每一列
print(np.min(a, axis = 1))
# [0.29046038 0.27287636]
print(np.max(a, axis = 0))
# [0.97312175 0.29046038 0.65062655 0.95807683]
