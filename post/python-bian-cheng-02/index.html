Python编程02：操作列表、if语句

2020-11-12

Python 编程02

第四章 ：操作列表

4.1 遍历整个列表

magicians = ['alice', 'david', 'carolina']
for magician in magicians:
# 冒号告诉Python，下一行时循环的第一行。
    print(magician)

每行缩进可以在for循环中包含更多操作。

4.3创建数值列表

使用函数range()可以轻松地生成一系列数

for val in range(1, 5):
    print(val)

Remark: 在这个示例中，range()只打印 1 ~ 4. （在5处停止）

使用list()将range()的结果转换成列表：

numbers = list(range(1, 6))
print numbers # [1, 2, 3, 4, 5, 6]

给range指定第三个参数可以指定步长：

even_numbers = list(range(2, 11, 2))

列表解析：首先指定一个描述性的列表名，然后再方括号中定义一个表达式，生成要存储到列表中的值。

squares = [value ** 2 for value in range(1, 11)]

4.4 列表切片

  • 处理列表的部分元素，Python称之为切片
  • 通过指定第一个元素和最后一个元素的索引实现
players = ['charles', 'martina', 'michael', 'florence', 'eli']
print(players[0 : 3])
# ['charles', 'martina', 'michael']

第一个索引缺省从头开始: print(players[ : 3])

第二个索引缺省到末尾结束： print(players[2 : ])

复制列表

friends_foods = my_foods[:]
# Wrong: friends_foods = my_foods
# 这两个变量会指向一个列表

4.5元组

  • 不可变的列表称为元组
  • 元组用圆括号标识
dimensions = (200, 50)

不能修改元组的元素，但是可以给存储元组的变量赋值

dimensions = (200, 50)
# do something
dimensions = (400, 100)

第五章：if语句

5.1 一个简单示例

cars = ['audi', 'bmw', 'subaru', 'toyota']
for car in cars:
    if car == 'bmw':
        print(car.upper())
    else:
        print(car.title())

5.2 条件测试

  1. 使用and检查多个条件
age_0 >= 21 and age_1 >= 21
(age_0 >= 21) and (age_1 >= 21)
  1. 使用or 检查多个条件
age_0 >= 21 or age_1 >= 21
  1. 检查特定值是否包含在列表中
>>> requested_toppings = ['mushrooms', 'onions', 'pineapple']
>>> 'mushrooms' in requested_toppings
True
  1. 检查特定值是否不包含在列表中
banned_users = ['andrew', 'carolina', 'david']
user = 'marie'

if user not in banned_users:
    print(f"{user.title()}, you can post a response if you wish.")

5.3 if语句

age = 12

if age < 4:
    print("Your admission cost is $0.")
elif age < 18:
    print("Your admission cost is $25.")
else:
    print("Your admission cost is $40.")

可以添加任意数量的elif代码块，Python并不要求if - elif 结构后面必须有else代码块。

  1. if - elif - else 仅仅适合用于只有一个条件满足的情况 ：遇到通过了的测试后，Python就跳过余下的测试。然而有时候必须检查你关心的所有事件。在这种情况下，应该使用一些列不包含elif 和 else 的代码块的简单if语句。
  1. 确定列表不是空的
requested_toppings = []

if requested_toppings :
    # do something
else:
    print("Are you sure you want a plain pizza?")
