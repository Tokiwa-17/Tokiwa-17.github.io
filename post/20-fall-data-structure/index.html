1.基本术语

​ 堂兄弟节点: 具有相同的祖父。

​ 节点A深度：**从根到A**所经过的边数。

​ 节点A高度：从A到叶子节点所经过边数的最大值

​ 树的高度: 树的根节点的高度。单节点树高度为0， 空节点树高度为-1.

height(T)≥height(v)+depth(v)height(T) \geq height(v) + depth(v)height(T)≥height(v)+depth(v)

​ 完全二叉树: 除了最后一层外，其余各层全是满的，并且最后一层的节点尽可能往左靠。第i层的最大节点数2i2 ^ i2i.

平衡二叉树: 树中任意节点的左右子树的高度差不超过k（通常k为1）

二叉树的性质:

  • 若二叉树的层次从0开始，则二叉树第i层最多有2i2 ^ i2i个节点。

  • 深度为k的二叉树最少有k + 1个节点，最多有2k+1−12 ^ {k + 1} -12k+1−1 个节点。

  • 对任一棵二叉树，若度为2**(指出度或分叉数)**的节点有n2n_2n2​个，则叶节点个数满足n0=n2+1.n_0 = n_2 + 1.n0​=n2​+1.

    Proof:

    n=n0+n1+n2.n = n_0 + n_1 + n_2.n=n0​+n1​+n2​.除根节点之外，每个节点有一个分支进入：B=n−1=n1+2n2.B = n - 1 = n_1 + 2n_2.B=n−1=n1​+2n2​.

  • 树的性质：树中节点数 = 总分叉数 + 1：

    ​ N=1+n1+2n2+...+knk.N = 1 + n_1 + 2n_2 + ... + kn_k.N=1+n1​+2n2​+...+knk​.

int BinTree<T>::updateHeight(BinNodePosi(T) x){

template <typename T>
void BinTree<T>::updateHeightAbove(BinNodePosi(T) x){
    while(x){
        updateHeight(x);
        x = x -> parent;
    }
}//O(depth(x))

template <typename T>
BinNodePosi BinTree<T>::insertAsLc(BinNodePosi x, T const &e){

    x -> insertAsLc(e);
    updateHeightAbove(x);

}

priority_queue<Node> q;
void dijkstra(int s){
    d[s] = 0;
    q.push(Node{s, 0});
    while(!q.empty){
        Node now = q.top();
        q.pop();
        int u = now.pos, dis = now.dis;
        if(vis[u]) continue;
        for(int i = head[u];i;i = E[i].next){
            int v = E[i].v;
            if(d[v] > d[u] + E[i].w){
                d[v] = d[u] + E[i].w;
                if(!vis[v]) q.push((Node){v, d[v]});
            }
        }
    }
}

SPFA——Bellman-Ford算法的优化

很多时候我们并不需要那么多无用的松弛优化，很显然，只有上一次被松弛的节点所连接的边，才有可能引起下一次的松弛操作。如果用队列来维护哪些节点可能会引起松弛操作，就能只访问必要的边了。

  • 如果图是随机生成的，时间复杂度O(KM)

  • 实际上SPFA的时间复杂度为O(NM),可以构造出卡SPFA的数据

拓扑排序

  • 拓扑排序必定保证G无环。

  • 拓扑排序不唯一。

  • Algorithm:

    1. 在AOV网络中选一个没有直接前驱的顶点，放入排序集中。
    2. 从图中删去该节点，同时删除所有它发出的有向边，更新邻点入度
    3. 重复上述过程，直到：
      • 全部节点输出，拓扑排序完成。
      • 图中还有未输出的节点（所有剩余节点入度均不为0），这是网络中必定存在有向环。
