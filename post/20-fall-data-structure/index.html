图的遍历

  • 遍历中组成树的一类边叫树边，与相应顶点构成遍历树。
  • 强调对特定状态顶点的查找，故称图搜索。
  • BFS、DFS, 均可在O(n + e)时间内完成。

BFS对应于层次遍历：越早访问的顶点，其邻域顶点越优先被访问。反复从队列中取出队头访问，并将其未被访问的邻域顶点入队。必须对顶点进行标记，已访问过的顶点不再入队。

struct Node{

vector<Node> Adj[MAXV];//Adj[u]存放u为始点连接的所有终点

int prim(){
    fill(d + 1, d + MAXV + 1, INF);
    d[1] = 0;
    int res = 0;
    for(int i = 0;i < n;i++){
		int u = -1, MIN = INF;
        for(int j = 1;j <= n;j++){
			if(!vis[j] && d[j] < MIN){
                u = j;
                MIN = d[j];
            }
        }
        if(u == -1) return -1;
        vis[u] = true;
        res += d[u];
        for(int j = 0;j < Adj[u].size();j++){
            int v = Adj[u][j].v;
            if(!vis[v] && Adj[u][v].dis < d[v])
                d[v] = Adj[u][v].dis;
        }
        return res;
    }
}

  • 边权值非负的单源最短路径问题

    最短路径树：

    • 单调性：最短路径的任意前缀也是最短路径。
    • 无环性

    Dijkstra: 首先求出长度最短的一条最短路径(s, u), 更新SPT的顶点集及到其他各点的最短距离。依次类推，指导所有顶点进入SPT集合。

堆优化的dijkstra:整体时间复杂度O(elogn).

il void addedge(int u, int v, int w){
    E[++cnt].v = v, E[cnt].w = w;
    E[cnt].next = head[u];
    head[u] = cnt;
}
priority_queue<Node> q;
void dijkstra(int s){
    d[s] = 0;
    q.push(Node{s, 0});
    while(!q.empty){
        Node now = q.top();
        q.pop();
        int u = now.pos, dis = now.dis;
        if(vis[u]) continue;
        for(int i = head[u];i;i = E[i].next){
            int v = E[i].v;
            if(d[v] > d[u] + E[i].w){
                d[v] = d[u] + E[i].w;
                if(!vis[v]) q.push((Node){v, d[v]});
            }
        }
    }
}

拓扑排序

  • 拓扑排序必定保证G无环。

  • 拓扑排序不唯一。

  • Algorithm:

    1. 在AOV网络中选一个没有直接前驱的顶点，放入排序集中。
    2. 从图中删去该节点，同时删除所有它发出的有向边，更新邻点入度
    3. 重复上述过程，直到：
      • 全部节点输出，拓扑排序完成。
      • 图中还有未输出的节点（所有剩余节点入度均不为0），这是网络中必定存在有向环。
