利用Python 进行数据分析

第二章:Python 语言基础

a = [1, 2, 3]
b = a

在Python中，a和b实际上指向了相同的对象。

将对象作为参数传递给一个函数时，指向原始对象的新的本地变量就会被创建而无需复制。

  • 了解对象的类型

isinstance接受一个包含类型的元组，你可以检查对象的类型是否是元组中存在的类型之一。

a = 5; b = 4.5
isinstance(a, (int, float))
# True
isinstance(b, (int,	float))
# True
  • 检查一个对象是否可以迭代

可以使用 iter函数检查一个对象是否实现了迭代器协议

def isiterable(obj):
    try:
        iter(obj)
        return True
    except TypeError: # 不可遍历
    	return False

isiterable('a string') # True
isiterable([1, 2, 3]) # True
isiterable(5) # False

if not instance(x, list) and isiterable(x):
    x = list(x)

Remark: 优先级 not > and > or

  • 运算

检查两个引用是否指向同一个对象，可以使用is关键字。

a = [1, 2, 3]
b = a
c = list(a) # list 函数总是创建一个新的列表

a is b # True
a is not c # True

is和is not的常用之处是检查一个变量是否为None

a = None

a is None # True
